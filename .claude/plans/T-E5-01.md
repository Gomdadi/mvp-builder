# T-E5-01 프로젝트 CRUD API 플랜

## Context

클라이언트가 파이프라인 각 단계 완료 후 분석 문서, 태스크 목록, 생성 파일을 조회할 수 있어야 한다.
세션 기반(no-auth) 구조이므로 클라이언트가 localStorage에 projectId를 저장하고 이를 통해 조회.

**엔티티 관계**
- `Project` (1) → (N) `AnalysisDocument`, `PipelineRun`
- `PipelineRun` (1) → (N) `Task`
- S3 키 패턴: `generated/{projectId}/{filePath}` — `s3.service.ts`의 `generatedKey()` 단일 진실 공급원

**전역 설정**: `setGlobalPrefix('v1')`, `ValidationPipe({ whitelist: true, forbidNonWhitelisted: true, transform: true })`

### 2. `src/projects/projects.service.ts`

**의존성 주입**
```typescript
constructor(
  @InjectRepository(Project) private readonly projectRepo,
  @InjectRepository(AnalysisDocument) private readonly analysisDocumentRepo,
  @InjectRepository(PipelineRun) private readonly pipelineRunRepo,
  @InjectRepository(Task) private readonly taskRepo,
  private readonly s3Service: S3Service,

```

**메서드별 구현**

- `createProject(dto)`: `projectRepo.create()` + `save()` → `{ id, name, status, createdAt }` 반환
- `getProject(id)`: `findOne({ where: { id } })` → 없으면 `NotFoundException`
- `getAnalysisDocument(id)`: `findOne({ where: { id } })` → 없으면 `NotFoundException`, `{ id, projectId, version, erd, apiSpec, architecture, isConfirmed, createdAt }` 반환
- `getPipelineRunTasks(pipelineRunId)`: pipelineRun 존재 확인 후 `taskRepo.find({ where: { pipelineRunId }, order: { orderIndex: 'ASC' }, select: ['id', 'name', 'type', 'orderIndex', 'status'] })`
- `getProjectFileTree(projectId)`: project 존재 확인 → `s3Service.listGeneratedFiles(projectId)` → `buildTree(paths)` 호출
- `getProjectFile(projectId, filePath)`: project 존재 확인 → `s3Service.downloadGeneratedFile(projectId, filePath)` → `{ path, content }` 반환

**`buildTree()` private 헬퍼**

S3 경로 배열(`["src/main.ts", "src/app/app.module.ts", "package.json"]`)을 재귀 트리로 변환:

```typescript
interface FileNode {
  name: string;
  path: string;
  children?: FileNode[];
}

private buildTree(paths: string[]): FileNode[] {
  const root: FileNode[] = [];
  for (const filePath of paths) {
    const parts = filePath.split('/');
    let current = root;
    let accumulated = '';
    for (let i = 0; i < parts.length; i++) {
      accumulated = accumulated ? `${accumulated}/${parts[i]}` : parts[i];
      let node = current.find((n) => n.name === parts[i]);
      if (!node) {
        node = { name: parts[i], path: accumulated };
        if (i < parts.length - 1) node.children = [];
        current.push(node);
      }
      if (node.children) current = node.children;
    }
  }
  return root;
}
```

### 3. `src/projects/projects.controller.ts`

```typescript
@Controller()  // prefix 없음 — 각 엔드포인트 prefix가 달라서
export class ProjectsController {
  @Post('projects')
  @HttpCode(HttpStatus.CREATED)
  createProject(@Body() dto: CreateProjectDto) { ... }

  @Get('projects/:id')
  getProject(@Param('id') id: string) { ... }

  @Get('analysis-documents/:id')
  getAnalysisDocument(@Param('id') id: string) { ... }

  @Get('pipeline-runs/:id/tasks')
  getPipelineRunTasks(@Param('id') id: string) { ... }

  @Get('projects/:id/files')
  getProjectFiles(
    @Param('id') id: string,
    @Query('path') filePath?: string,
  ) {
    if (filePath) return this.projectsService.getProjectFile(id, filePath);
    return this.projectsService.getProjectFileTree(id);
  }
}
```
